Overview


This class focuses on learning the concepts of computer system and network security. Specifically, we will explore several aspects of computer security with the goal of defending complex computer systems. And we believe that the best way to defend a system is to learn how to break it! Thus, we will cover some of the major attack/defense techniques in a number of topics in system and network security. The class will help students gain hands-on experience in security through programming assignments and a final project.

Check the course Syllabus for more information.
